YouTube Video Downloader

A user-friendly web application that allows users to download YouTube videos by simply pasting a video link. The app processes the request using Django on the backend and delivers downloadable formats with the help of the yt-dlp library.

Technologies Used: Django, HTML, JavaScript, yt-dlp
Key Features: URL input for downloading videos, dynamic response with download options, and clean user interface.
What I Learned: Integrating Python libraries with web interfaces, handling user input securely, and enhancing the user experience through responsive design.
